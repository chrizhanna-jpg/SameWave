SameWave — Windows local AAB
===========================

ONE upload file after a successful build:

  C:\w\app\aab\SameWave-latest.aab

Build command (from your repo):

  cd artifacts\same-same
  pnpm run build:aab:local

The script refuses to finish unless that file exists with today's date.

First-time setup (if C:\w\app is missing):

  pnpm run setup:short-path-build
