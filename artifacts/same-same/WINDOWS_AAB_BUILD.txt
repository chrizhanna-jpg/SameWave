SameWave — Windows local AAB (same as vc43 build)
================================================

1. Bump versionCode in app.json (already done in repo)
2. From repo checkout:
     cd artifacts\same-same
     pnpm run build:aab:local
3. Upload:
     C:\w\app\aab\SameWave-vc47.aab

This is the identical workflow used for SameWave-vc43.aab:
  sync repo -> C:\w\app, Gradle bundleRelease, copy to C:\w\app\aab\
